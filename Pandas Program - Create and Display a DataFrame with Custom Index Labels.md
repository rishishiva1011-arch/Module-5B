---

## ðŸ’» Program

```
import pandas as pd
import numpy as np
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [24, 27, 22],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
label = np.array(['A', 'B', 'C'])
df = pd.DataFrame(data, index=label)
print("DataFrame with Custom Index Labels:")
print(df)
```
